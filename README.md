# HOLYSTONE Construction Company Website

A modern, responsive website for HOLYSTONE Construction Company, showcasing residential construction projects and services. Built with clean HTML, CSS, and JavaScript, featuring smooth animations and a professional design.

## 🏗️ About

HOLYSTONE Construction Company specializes in residential construction, transforming dreams into beautiful living spaces. With years of expertise and unwavering commitment to quality, we deliver homes that stand the test of time.

**Leadership:**
- **Deepak Sadanand Hanchate** - Chairman & M.D.
- **Dheeraj Sadanand Hanchate** - Director

## ✨ Features

- **Responsive Design**: Fully responsive layout that works seamlessly on all devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Project Showcase**: Display completed and ongoing construction projects
- **Contact Integration**: Easy-to-use contact form and direct communication links
- **Smooth Animations**: GSAP-powered animations for enhanced user experience
- **Custom Typography**: Beautiful custom fonts (ClashDisplay & Satoshi)
- **Interactive Navigation**: Mobile-friendly hamburger menu

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with custom properties and animations
- **JavaScript**: Interactive functionality
- **GSAP (GreenSock Animation Platform)**: Smooth scroll animations and transitions
- **Font Awesome**: Icon library
- **Custom Fonts**:
  - ClashDisplay (Display font)
  - Satoshi (Body font)
